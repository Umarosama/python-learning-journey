# 🎮 Rock, Paper, Scissors

## 📌 Overview

A command-line implementation of the classic Rock, Paper, Scissors game built in Python.

The player selects Rock, Paper, or Scissors, while the computer randomly generates its own choice. The program compares both choices using conditional statements and announces the winner.

---

## ✨ Features

- Player vs Computer gameplay
- Random computer choice
- Case-insensitive user input
- Input validation
- Winner determination
- Clean function-based structure

---

## 🧠 Concepts Practiced

- Functions
- Function parameters
- Return values
- User input
- Dictionaries
- Conditional statements (`if`, `elif`, `else`)
- Nested conditionals
- String methods (`.lower()`)
- `random.choice()`
- f-strings

## 🚀 How to Run

1. Clone the repository.

```bash
git clone <repository-url>
```

2. Navigate to the project folder.

```bash
cd Rock-Paper-Scissors
```

3. Run the program.

```bash
python rock_paper_scissors.py
```
