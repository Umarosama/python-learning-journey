# Elevator Floor Converter

## Overview

This is a beginner Python project created while learning Python fundamentals.

The program converts floor numbering between the United States and Europe.

Since Europe considers the Ground Floor as **0**, while the United States considers it **1**, the program adjusts the entered floor accordingly.

---

## Concepts Practiced

- Variables
- Input / Output
- Type Conversion
- Expressions
- Arithmetic Operators
- Conditional Statements (`if` / `else`)

---

## Example

Input

Enter the floor number: 5
You are in the USA or Europe? USA

Output

The corresponding floor in Europe is:
4

---

## Learning Goal

This project was built to practice basic Python syntax and understand how user input, variables, arithmetic operations, and conditional logic work together in a real program.
